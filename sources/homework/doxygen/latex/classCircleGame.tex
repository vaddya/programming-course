\subsection{Подробное описание}
Круговая игра 

По кругу располагаются num\+\_\+of\+\_\+players человек. Ведущий считает по кругу, начиная с первого, и выводит num\+\_\+for\+\_\+kick-\/го человека. Круг смыкается, счет возобновляется со следующего. Так продолжается, пока в круге не останется только один человек. Найти номер этого человека.
